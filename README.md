# ArnoldBot

This is the first discord bot that I created: Arnold Bot. Currently a WIP.

# Commands:

!help, states commands to be entered by users
!hello, the bot gives a greeting message
!play, plays the youtube URL following this command

# Automatic Functions:

DadBot, replies with a dad joke when a user types "I'm" + a message after
LanguageFilter: filters through a list of words that are banned and removes the message
